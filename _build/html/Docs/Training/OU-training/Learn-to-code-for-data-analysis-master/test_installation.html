This Jupyter notebook is for The Open University‚Äôs Learn to code for data analysis course. It checks whether your software installation is working as expected.

To do that, you must open this notebook from within the Jupyter app in Anaconda or by using a cloud service like CoCalc or Microsoft Azure.

Then, in the menu above, click on ‚ÄòCell‚Äô and then on ‚ÄòRun all‚Äô. This will execute the code below. (The code will be explained in the course.)

Anaconda#

If you‚Äôre using Anaconda, if the pandas version displayed is 0.20 or higher and you don‚Äôt get any error messages, then all‚Äôs well and you can close this notebook (menu ‚ÄòFile‚Äô, option ‚ÄòClose and Halt‚Äô) and then delete the notebook from your disk if you wish.

If there is an error message saying you don‚Äôt have Python 3, or if the pandas version displayed is below 0.20, then one of the following has probably happened:

  • you installed the wrong variant of Anaconda (there are two variants, for Python 2 and Python 3, of the same version of Anaconda);

  • during the installation you decided to keep your existing Python installation.

To see which software versions you have installed, click on the ‚ÄòHelp‚Äô menu above, and then on the ‚ÄòAbout‚Äô option. You should have the notebook server version 5.0 or higher, Python version 3.5 or higher, and IPython version 6.1 or higher.
